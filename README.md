# Library Management System (SQL + PL/SQL)

##  Description
A database project simulating a library system. Includes tables for authors, books, members, and loans, plus stored procedures and triggers to handle borrowing and returning books.

##  Contents
- `schema.sql`: Database schema with constraints.
- `data.sql`: Sample records.
- `queries.sql`: Example queries (top borrowed books, overdue loans).
- `procedures.sql`: Procedures for loan and return operations.
- `triggers.sql`: Automations for loan status.

##  How to Run
1. Import `schema.sql` into PostgreSQL or Oracle.
2. Run `data.sql` to populate sample data.
3. Test queries in `queries.sql`.
4. Use procedures to simulate loan operations.

##  Skills Demonstrated
- Relational database design.
- PL/SQL with error handling.
- Reporting queries and views.
